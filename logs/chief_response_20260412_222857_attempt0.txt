```python
# 伪代码：两阶段残差包含法 (2SRI) 校正测量误差
class MeasurementErrorCorrection:
    def __init__(self, exposure, outcome, covariates, instruments):
        self.exposure = exposure  # 处方记录 (有偏)
        self.outcome = outcome
        self.ivs = instruments    # 遗传变异

    def fit(self):
        # 第一阶段：预测真实暴露
        # X_hat = f(Z, W)
        model_stage1 = LinearRegression()
        model_stage1.fit(self.ivs, self.exposure)
        exposure_residual = self.exposure - model_stage1.predict(self.ivs)

        # 第二阶段：预测结局，同时包含残差项以校正偏倚
        # Y = beta * X_hat + gamma * residual + error
        model_stage2 = LinearRegression()
        # 注意：2SRI在非线性模型（如Logit）中优于2SLS
        features = np.column_stack([self.exposure, exposure_residual, covariates])
        model_stage2.fit(features, self.outcome)
        return model_stage2.coef_
```

```python
class TrajectoryEncoder(nn.Module):
    def __init__(self, feature_dim, d_model, n_heads):
        super().__init__()
        self.embedding = TemporalEncoding(feature_dim, d_model)
        self.encoder_layer = nn.TransformerEncoderLayer(d_model=d_model, nhead=n_heads)
        self.encoder = nn.TransformerEncoder(self.encoder_layer, num_layers=4)

        x = x.permute(1, 0, 2) # Transformer needs [seq, batch, dim]
        traj_repr = self.encoder(x)
```

**边际结构模型（MSM）与IPW**：
1.  **构建治疗机制**：在每个时间点$t$，计算倾向性评分 $PS_t = P(A_t | \bar{A}_{t-1}, \bar{L}_t)$，其中$A$是PA，$L$是混杂变量（如之前的疾病史）。
2.  **计算稳定权重**：$SW_t = \prod_{k=1}^t \frac{P(A_k)}{P(A_k | \bar{A}_{k-1}, \bar{L}_k)}$。
3.  **加权伪队列**：使用$SW_t$对数据进行加权，使得加权后的样本中，PA的分布与过去的病史无关（模拟RCT）。
4.  **因果推断**：在加权后的数据上拟合逻辑回归或Cox模型，估计长期的平均处理效应（ATE）。

## 【假设 3：跨界盲区模式 - 物理建模驱动】

### 假设标题
解耦分辨率伪影与病理改变：基于物理信息神经网络（PINN）的小动物PET成像因果推断

### 裂缝模式
跨界盲区

### 学术三要素评估标准 (Academic Three Elements)
- **新颖性**：将**计算物理学**中的物理信息神经网络引入医学图像分析。论文1仅模拟了参数影响，未解决“图像质量差异被误诊为病理改变”的因果难题。
- **严谨性**：不使用有监督的“标签”，而是将**泊松噪声模型**和**点扩散函数（PSF）**作为物理约束嵌入网络损失函数。
- **颠覆性**：从“提高分辨率”转向“解耦物理效应”。不仅仅是让图更清楚，而是回答“图像上的这个高代谢区，是肿瘤的真实生物学特性，还是探测器分辨率不足造成的部分容积效应（PVE）？”

### 推演草稿 (Internal Reasoning)

**跨界盲区分析**：
在论文1（Brain PET imaging）的研究中，探讨的是参数优化。但这忽略了一个临床诊断中的致命盲区：
- **物理假象**：在脑PET或小动物PET中，由于**部分容积效应**，皮层表面的高代谢往往会“溢出”到脑脊液或低代谢区域。
- **误诊风险**：放射科医生或AI算法可能将这种物理导致的“信号模糊”误判为“浸润性生长”或“水肿”。
- **传统AI的缺陷**：目前的CNN超分辨率算法通常使用像素损失（MSE），这导致生成的图像过于平滑，丢失了医学上关键的纹理信息（异质性）。

**我的切入点 - PINN（Physics-Informed Neural Networks）**：
- 引入流体力学和光学中的PINN概念。
- 我们不训练网络去“猜”像素值，而是训练网络去学习**符合物理规律的真值**。
- **反因果问题**：观测图像 $Y$ 是真实图像 $X$ 与模糊核 $K$ 的卷积 + 噪声。我们要做的是**反卷积**，但使用神经网络作为正则化项。

### 核心科学问题（250字）

高分辨率PET成像研究（如论文1）存在一个未被解决的“因果归因”难题。当我们在图像上观察到“代谢活性的弥散”时，这究竟是源于生物学上的肿瘤侵袭性（真阳性），还是源于探测器的**部分容积效应**和散射导致的物理伪影（假阳性）？传统的迭代重建算法（如OSEM）仅能最大化似然估计，无法分离这两者。深度学习超分辨率方法虽然能提升PSNR（峰值信噪比），但会产生幻觉纹理，且不遵守放射性衰变的物理规律。本研究的核心问题是：如何利用**物理信息神经网络（PINN）**，将PET成像的物理模型（泊松统计、点扩散函数PSF）作为硬约束嵌入网络，从数学上解耦“物理因素”与“生物病理因素”，还原真实的体内放射性分布？

### 创新假设（220字）

我假设：PET图像的退化过程是一个可微的物理过程，可以用神经网络进行逆向求解。（1）构建一个双流网络，一路学习**物理退化参数**（分辨率、噪声水平），另一路还原**真实生物学分布**；（2）通过引入物理守恒定律（如局部区域内总放射性守恒），消除深度学习的幻觉伪影；（3）具体假设：在脑小胶质细胞激活的检测中，PINN解耦后的图像能够区分出“真实的边缘浸润”（代谢梯度符合生物学规律）和“PVE伪影”（梯度符合高斯扩散规律），使得AUC从0.82提升至0.94。

### 技术路线（650字）

**数据与物理建模**：
1.  **仿真数据**：使用GATE（Geant4 Application for Tomographic Emission）构建高保真蒙特卡洛模型，生成“Ground Truth”（真实生物分布）和“Observed Data”（模拟探测器数据）。
2.  **物理模型**：
    -   **观测方程**：$Y = Poisson( H X + B + S )$，其中H是系统矩阵（含PSF），B是背景，S是散射。
    -   **正则化项**：全变分（TV）约束或基于解剖学（MRI/CT）的先验约束。

**PINN架构设计**：
```python
# 伪代码：物理信息解耦网络
class PhysicsInformedPET(nn.Module):
    def __init__(self):
        self.generator = UNet() # 生成推测的真实图像 X_est
        self.psf_estimator = CNN() # 估计系统的PSF参数

    def forward(self, y_measured):
        X_est = self.generator(y_measured)
        # 1. 数据一致性项：重建的图像经过物理模型后应接近观测值
        y_reproj = self.physics_model(X_est) # 前向投影 + 模糊 + 噪声
        loss_data = mse(y_measured, y_reproj)

        # 2. 物理守恒项：总放射性守恒（仅在无噪声情况下）
        # 实际上用泊松似然：Sum(y_measured * log(y_reproj) - y_reproj)
        loss_physics = poisson_nll(y_measured, y_reproj)

        # 3. 解耦正则化项：图像梯度应主要由解剖结构决定，而非噪声
        loss_anatomy = texture_loss(X_est, anatomical_prior)

        return loss_data + loss_physics + loss_anatomy
```

### 统计学新颖性（Statistical Novelty）（200字）

本研究的统计学突破在于**物理先验的贝叶斯融合**。（1）**物理一致性损失**：传统的图像重建优化最大似然，我们引入泊松分布的对数似然作为损失函数，这是统计学上对噪声分布的正确假设，比L2 Loss（高斯假设）更符合PET的物理本质。（2）**偏差-方差分解的解耦**：通过将物理模型显式化，我们可以将预测误差分解为“建模误差”（物理模型不准）和“随机误差”（噪声），从而定量评估算法的稳健性。（3）**不确定性量化**：使用贝叶斯PINN（B-PINN）输出像素级的不确定性图，告诉医生哪些区域的“高代谢”是可靠的，哪些是受PVE干扰严重不可信的。

### 预期突破（280字）

**旧方法局限性**：

**我的颠覆性**：

2. **定量恢复**：不仅视觉上清晰，而且恢复了被模糊掉的绝对SUV值，这对于定量PET（如监测治疗效果）至关重要。
3. **无需成对训练**：利用物理模型，可以用非配对数据进行训练，解决了医学成像中金标准缺失的难题。

### 临床/转化价值（220字）

本研究将革新核医学的诊断标准：
