```python
class DeepSCM:
    def __init__(self, X_pre, Y_pre):
        # X_pre: 干预前的协变量矩阵 [N_samples, N_features]
        # Y_pre: 干预前的结果轨迹
        self.optimizer = ConvexOptimizer()

    def fit_synthetic_control(self, treated_unit_id, donor_pool):
        # 1. 特征降维与选择
        # 使用 Elastic Net 选择关键预测变量
        selected_features = ElasticNet_Selection(X_pre, alpha=0.01)
        
        # 2. 优化权重向量 W
        # 约束条件：权重和为1，且非负
        # 目标函数：最小化干预前的误差 ||X_treated - W*X_donors||
        W = self.optimizer.solve(
            objective=lambda w: mse(X_treated[selected], w @ X_donors[selected]),
            constraints=[sum(w) == 1, w >= 0]
        )
        
        # 3. 反事实模拟
        # 将权重 W 施加于干预后的 Donor 数据
        Y_counterfactual = W @ Y_donors_post
        
        # 4. 计算处理效应 (ATT)
        ATT = Y_treated_actual - Y_counterfactual
        return ATT, W
```

  - **特异成分**：特定亚型的特征（如偏头痛的搏动性特征）。

**算法核心**：
使用**解耦变分自编码器**框架，但编码器替换为脉冲神经元。利用SNN的时间稀疏性，我们可以精确地定位信号中的**因果事件**（如某个特定的尖峰模式），从而实现高精度的分类和亚型分层。

### 核心科学问题（240字）

当前的神经形态计算（如论文1的NIR）主要关注软硬件互操作性，忽略了生物信号内部的异质性结构。在临床神经科学（如头痛研究）中，EEG/MEG信号具有极高的时间分辨率和非平稳性，传统的深度学习方法（RNN/LSTM）将其视为连续值序列，丢失了神经放电的**事件驱动特性**。本研究的核心问题是：如何利用SNN的生物保真性，将高维的神经形态信号解耦为**“共享特征”**和**“个体/亚型特征”**？通过这种解耦，我们能否构建一个跨中心、跨设备的通用临床辅助诊断模型，解决不同医院设备差异带来的分布偏移问题？

**数据与预处理**：
1. **数据源**：TuH EEG Corpus（公开癫痫/头痛数据，包含多中心数据）。
2. **NIR转换**：将原始连续EEG信号转换为脉冲序列（Latency Coding 或 Rate Coding），符合论文1的中间表示标准。

**算法架构：解耦脉冲神经网络**：
```python
# 伪代码：解耦SNN框架
class DisentangledSNN(nn.Module):
    def __init__(self):
        # 使用LIF神经元作为基本单元
        self.snn_encoder = LIF_Layer(threshold=1.0)
        # 解耦瓶颈
        self.z_content = Variable(latent_dim=64)  # 共享特征（如疼痛强度）
        self.z_style = Variable(latent_dim=64)   # 特异特征（如亚型、设备噪声）

    def forward(self, x_spikes):
        # 1. 时间卷积与脉冲积分
        mem = self.snn_encoder.integrate(x_spikes)
        
        # 2. 变分解耦
        # 将膜电位映射到两个独立流形
        mu_content, logvar_content = self.encoder_content(mem)
        mu_style, logvar_style = self.encoder_style(mem)
        
        # 3. 重参数化 trick
        z_content = reparameterize(mu_content, logvar_content)
        z_style = reparameterize(mu_style, logvar_style)
        
        # 4. 解码与分类
        # 仅使用 z_content 进行分类（提高鲁棒性）
        # 结合 z_style 生成信号（用于重建）
        y_pred = self.classifier(z_content)
        x_recon = self.decoder(z_content, z_style)
        
        return y_pred, x_recon
```

**训练策略**：
1. **跨模态对比损失**：最大化同一患者不同时间段/不同导联的 $z_{content}$ 相似度，最小化 $z_{style}$ 的互信息。
2. **分布外（OOD）泛化**：在训练时故意丢掉某些中心的数据，验证模型在未见设备上的表现。
3. **因果干预**：在推理阶段，固定 $z_{style}$，仅改变 $z_{content}$，观察输出变化，验证因果性。

### 统计学新颖性（Statistical Novelty）（180字）

本研究的统计学突破在于将**因果表征学习**引入神经形态计算。（1）**分布外泛化**：通过显式建模 $z_{style}$（通常代表混杂因素，如设备差异），我们实现了 $P(Y|X)$ 在分布偏移下的不变性。这在统计学上等同于**协变量偏移校正**。（2）**独立成分分析（ICA）的扩展**：传统ICA假设线性混合，而我们的SNN框架假设**非线性动力学混合**，更符合神经信号的传播特性。（3）**变分推断的贝叶斯先验**：我们对潜在变量施加了统计先验（如高斯先验），使得模型具有正则化效果，防止在小样本医疗数据上过拟合。
