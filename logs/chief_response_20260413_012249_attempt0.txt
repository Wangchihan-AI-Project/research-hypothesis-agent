**分层贝叶斯测量误差模型**：
我们将每个患者 $i$ 的真实药物暴露状态 $E_i^*$ 视为隐变量，观测到的处方 $R_i$ 和自述 $S_i$ 视为带有不同误差率的观测值。

```stan
// Stan Pseudo-code
data {
  int N; // 患者数量
  int script[N]; // 处方记录 (0/1)
  int report[N]; // 自述 (0/1)
  int outcome[N]; // 结局 (如流产)
  vector[N] SES; // 社会经济地位
}
parameters {
  vector[N] E_true; // 真实暴露概率
  real<lower=0, upper=1> sens_script; // 处方灵敏度
  real<lower=0, upper=1> spec_report; // 自述特异度
  real beta; // 暴露对结局的因果效应
}
model {
  // 先验
  E_true ~ beta(1, 1);
  sens_script ~ beta(10, 1); // 处方记录通常很灵敏
  spec_report ~ beta(5, 5);  // 自述特异度受SES影响
  
  // 似然：观测数据是由真实暴露生成的
  script ~ bernoulli(E_true * sens_script + (1-E_true)*(1-sens_script));
  report ~ bernoulli(E_true * spec_report + (1-E_true)*(1-spec_report));
  
  // 因果模型：结局由真实暴露驱动
  outcome ~ bernoulli_logit(intercept + beta * E_true + gamma * SES);
}
```

class BioPSF_GNN(nn.Module):

        # 生物PSF是一个可学习的层，输入是组织密度和基因表达（预测溶酶体活性）

    def forward(self, raw_pet, tissue_density_map, lysosomal_gene_exp):
        # 1. 传统物理模糊
        phys_blur = self.physical_psf(raw_pet)
        
        # 2. 估计生物学模糊

### 技术路线（600字）

```python
# 伪代码：GNN for MNAR Imputation
class MissingGNN(nn.Module):
    def __init__(self):
        self.encoder = GraphSAGE(in_channels=num_features, hidden_channels=64)
        self.imputer = MLP(in=64, out=num_features)
        self.discriminator = MLP(in=64, out=1) # 判别缺失机制

    def forward(self, graph, x_obs, mask):
        # 1. 编码观测特征和缺失模式
        # x_obs: 初始均值填充
        # mask: 0=missing, 1=observed
        h = self.encoder(graph, x_obs)
        
        # 2. 预测真实值 (Outcome Model)
        x_imputed = self.imputer(h)
        
        # 3. 预测缺失概率 (Mechanism Model - 用于识别MNAR)
        prob_missing = self.discriminator(h)
        
        return x_imputed, prob_missing
```

**模式混合模型**：
构建 Heckman 校正模型：
1.  **选择方程**：Probit模型预测数据是否缺失（$R=1$）。
2.  **结果方程**：预测药物暴露量。
3.  **关键步骤**：引入 Inverse Mills Ratio (IMR) 来校正选择性偏倚。

**不确定性量化**：
使用**蒙特卡洛EM算法**估计参数，并利用Rubin Rules合并100次插补数据集的结果，给出包含缺失机制不确定性的置信区间。

- **严谨性**：计算**自然直接效应（NDE）**与**自然间接效应（NIE）**，量化溶酶体在药物起效过程中的贡献比例。

### 推演草稿

- **文献3**关注药物进入溶酶体。

**跨界引入 - 因果中介**：
- 我们可以利用**Baron & Kenny**的现代版——**Imai-Keele因果中介框架**。
- **问题**：药物（$T$）导致细胞死亡（$Y$）。
- **中介**：药物改变溶酶体pH（$M$），导致酶释放（$Y$）。
- **分析**：
    - 如果阻断溶酶体pH变化（即切断路径 $T \to M$），药效下降多少？这代表**NIE（间接效应）**。
    - 如果不经过溶酶体，药物直接作用于靶点，这是**NDE（直接效应）**。

**假设**：
对于某些化疗药（如柔红霉素），**NIE > 60%**。这意味着这些药的主要作用机制其实是"破坏溶酶体"而非"抑制拓扑异构酶"。这将重写药理学教科书。
