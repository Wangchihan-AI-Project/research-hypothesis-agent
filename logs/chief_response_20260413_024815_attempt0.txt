**跨界盲区分析**：
Paper 1 研究的是Hf2C/HfSe2体系。这里存在一个经典的物理/化学难题：
- **应变**：晶格失配引入的应变会改变能带结构。
- **掺杂**：碳原子引入可能会提供电子（n型掺杂）。
这两者都会导致**肖特基势垒高度（SBH）下降**。
在实验中，我们制造了一个Hf2C/HfSe2器件。我们看到了SBH下降。
**但是**，是因为应变？还是因为掺杂？还是两者都有？
在湿实验中，你很难制造一个“有应变但无掺杂”的对照组，因为碳原子的引入必然带来这两者。
这就是**因果推断中的“混杂因子”**问题。

**为什么合成控制法是跨界盲区**：
- **计量经济学解法**：在评估某个政策（如出台新税法）的效果时，我们无法让历史倒流。但我们可以构建一个**“合成城市”**，这个合成城市是由其他未实施该政策的城市加权组合而成的，它完美模拟了目标城市在“未实施政策”情况下的表现。
- **本研究的解法**：我们可以构建一个**“虚拟材料”**。
  - 我们想要研究Hf2C/HfSe2（处理组）。
  - 我们可以从Materials Project数据库中，找到一组没有碳化但晶格常数相似的化合物（如HfS2, ZrSe2等）。
  - 通过加权组合这些材料的电子结构特征，构建一个**“合成HfSe2”**，它模拟了HfSe2在“只受应变、不受掺杂”情况下的表现。
  - 对比真实的Hf2C/HfSe2与合成HfSe2，其差值就是**化学掺杂的纯因果效应**。

### 核心科学问题（280字）

材料科学中存在严重的**“对照组缺失”**导致的归因模糊。Paper 1 观察到的低势垒是应变与掺杂的混合效应，无法区分物理机制（应变调制）与化学机制（电荷转移）。传统的对比实验（如更换材料）难以控制单一变量，因为材料的改变往往同时引入多种物理化学性质的变化。本研究核心问题是：如何利用**合成控制法**，从海量已知的材料数据库中构建一个**“反事实材料”**作为完美对照，从而在单样本实验层面剥离**外延应变**与**化学掺杂**对界面态的混杂影响？

### 创新假设（220字）

我假设：任何复杂的异质界面体系都可以通过**稀疏投影**分解为一组基础材料特征的线性组合。具体假设为：（1）对于Hf2C/HfSe2体系，可以通过L1-正则化回归，从MP数据库中筛选出5-8种“基材料”（如HfS2, ZrSe2, TiSe2等），其加权组合的能带结构在无碳化情况下与HfSe2高度拟合（R² > 0.95）；（2）将真实体系的SBH减去合成体系的SBH，得到的**差异量**即为“化学掺杂的净因果效应”；（3）这种因果效应在空间分布上是不均匀的，通过**SCM (Synthetic Control Method) 空间映射**可以发现掺杂效应主要集中在界面边缘1-2 nm范围内。

### 技术路线（650字）

**数据构建**：
1. **特征库建立**：从Materials Project和OQMD数据库中提取5000种过渡金属硫族化合物（TMDs）的特征。
   - 特征向量：[晶格常数a, 晶格常数c, 功函数, 电子亲和能, 弹性模量, 介电常数]。
2. **处理组数据**：Paper 1的Hf2C/HfSe2的DFT计算结果（能带、态密度、SBH）。

**合成控制模型**：
```python
# 伪代码：合成控制法
class SyntheticControl:
    def __init__(self, donor_pool):
        self.donor_pool = donor_pool  # 候选对照材料库

    def fit(self, treated_unit_features, pre_period_features):
        # 优化目标：最小化处理组在"处理前"的特征误差
        # 这里"处理前"指未发生碳化/反应前的HfSe2本征特征
        objective = minimize(
            lambda W: mse(treated_unit_features, W @ donor_pool_features),
            constraints={sum(W) == 1, W_i >= 0}  # 约束：权重和为1，且非负
        )
        self.W_opt = objective.x  # 最优权重向量
        self.synthetic_unit = self.W_opt @ self.donor_pool_outcomes

    def causal_effect(self, treated_outcome):
        # 处理效应 = 真实结果 - 合成结果
        return treated_outcome - self.synthetic_unit
```

**因果效应量化**：
1. **时间维度验证**：如果有多时间点的实验数据（如原位加热），可以验证“合成对照”在处理前（未碳化）的拟合优度。
2. **安慰剂检验**：
   - 随机选取一个未处理的材料，假装它是处理组，重复SCM流程。
   - 如果随机材料的效果显著小于Hf2C/HfSe2，则证明原结论不是随机巧合。
3. **敏感性分析**：计算**Rosenbaum边界**，评估潜在的未测量混杂（如表面缺陷）对结论的影响。

**物理机制解析**：
1. 分离出的**应变效应**（由合成对照代表）。
2. 分离出的**掺杂效应**（由差值代表）。
3. 将掺杂效应映射到实空间，分析其对MIGS波函数衰减长度的具体影响。

### 统计学新颖性（200字）

本研究首次将**合成控制法**这一政策评估的黄金标准引入材料科学。（1）**数据驱动的对照构建**：突破了物理实验难以制备完美对照组的限制，利用大数据构建“虚拟材料”作为基准。（2）**加权平均因果效应**：不同于简单的“单点对比”，SCM通过高维特征匹配，确保对照材料在除“目标变量（如碳化）”以外的所有维度上都与处理组高度一致。（3） **推断有效性**：通过安慰剂检验和留一法，严格验证因果推断的统计学显著性，排除了偶然性发现。

### 预期突破（280字）

**旧方法局限性**：
- 简单对比：对比HfSe2和HfS2，但两者化学性质不同，无法剥离变量。
- 经验估计：凭经验猜测“主要是应变”或“主要是掺杂”，缺乏量化证据。
- 理论计算：试图通过计算特定缺陷模型来模拟，但模型过于理想化。

**我的颠覆性**：
1. **混杂剥离**：首次量化了“化学掺杂”对接触电阻的独立贡献（例如：30%来自应变，70%来自掺杂）。
2. **虚拟实验**：在不进行真实合成的情况下，预测“如果只加应变不加碳”会发生什么。
3. **通用工具**：建立了一个通用的“材料因果效应评估框架”，可应用于电池、催化等其他材料体系。

### 临床/转化价值（230字）

本研究为材料设计提供**因果图谱**：
1. **精准改性**：明确告诉工程师，如果目标是降低势垒，应该优先调节“掺杂浓度”而非单纯“施加应变”。
2. **失效分析**：如果器件性能下降，可以通过SCM快速定位是“应变弛豫”还是“杂质扩散”导致。
3. **知识产权**：基于因果效应的发现，可以申请更精准的材料成分专利，保护范围从“含碳”细化为“特定掺杂比例”。
