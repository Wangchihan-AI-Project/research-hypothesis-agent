- **分层贝叶斯**：假设每个患者的空间参数来自于一个总体分布。这允许我们“借用信息”：对于数据质量差的患者，模型会将其参数“收缩”向总体均值，从而得到更稳健的估计。

```python
# 伪代码：HB-GNN 模型
import torch
import pyro

class HierarchicalSpatialGNN(pyro.nn.PyroModule):
    def __init__(self, node_dim, edge_dim, n_patients):
        # 共享参数：空间互作的通用规则
        self.gnn_conv = GNNLayer(node_dim, edge_dim)
        # 超先验：患者参数的分布
        self.patients_hyper_prior = pyro.distributions.Normal(0, 1)
        
    def forward(self, graph_data, patient_id):
        # 1. GNN 提取空间特征
        spatial_features = self.gnn_conv(graph_data.x, graph_data.edge_index)
        
        # 2. 患者特异性随机效应
        # 每个患者拥有自己的权重矩阵 W_p ~ N(mu, sigma)
        with pyro.plate("patients", n_patients):
            W_patient = pyro.sample("W_patient", self.patients_hyper_prior).to(device)
            
        # 3. 个性化预测
        # Logits = Shared_Spatial_Features * Patient_Weights
        prediction = torch.matmul(spatial_features, W_patient[patient_id])
        return prediction
```

**跨界盲区分析**：
在肿瘤免疫学中，研究者习惯于比较“肿瘤内”与“基质内”的免疫细胞功能。

计量经济学在评估政策效果（如就读大学对收入的影响）时，为了解决“自我选择偏倚”，发展了PSM方法。
我们可以将“物理位置（肿瘤 vs 基质）”视为一种“处理”。
我们需要找到两组具有相同特征（克隆型、转录状态、细胞周期）的免疫细胞，一组在肿瘤内，一组在基质内，然后比较它们的杀伤功能。

### 核心科学问题（250字）

肿瘤免疫微环境研究中存在严重的**自我选择偏倚**。现有的比较分析（如比较“肿瘤浸润淋巴细胞TILs”与“外周淋巴细胞”或“基质淋巴细胞”的功能差异）忽略了细胞位置的非随机分配。具体而言，具有高细胞毒潜力的T细胞可能更倾向于迁移至肿瘤核心，或者相反，肿瘤核心的恶劣环境筛选出了特定的耐药亚克隆。如果我们直接比较转录组差异，得到的“差异表达基因”可能反映的是细胞的**内在异质性**，而非**微环境诱导的适应性改变**。本研究的核心问题是：如何利用**倾向性评分匹配（PSM）**消除细胞固有的转录本差异，从而分离出空间位置对细胞状态的**纯因果效应**？

### 创新假设（210字）

我假设：空间位置对T细胞耗竭的影响被严重高估，这主要归因于**选择性迁移**。具体假设为：（1）**零假设**：在匹配了TCR克隆型、细胞分化状态和代谢特征后，肿瘤核心与边缘的T细胞在耗竭标志物（如PDCD1, LAG3）表达上无显著差异；（2）**因果效应**：空间位置的因果效应仅存在于特定的代谢通路（如缺氧诱导因子HIF1a），而非常规的耗竭通路；（3）**反事实预测**：如果将基质中的T细胞“移植”入肿瘤核心（反事实），其状态改变将主要由缺氧驱动，而非抗原负荷驱动。

### 技术路线（550字）

**数据准备**：
1.  **多模态数据整合**：整合scRNA-seq（获取细胞状态）和空间转录组（获取位置坐标）。
2.  **构建数据集**：每个细胞是一个样本，协变量（X）包括基因表达谱（降维后）、TCR克隆型；处理变量（T）是位置（Core=1, Stroma=0）；结局变量（Y）是耗竭评分。

**倾向性评分匹配流程**：
```python
# 伪代码：PSM 流程
from sklearn.linear_model import LogisticRegression
from sklearn.neighbors import NearestNeighbors

# 1. 估计倾向性评分
# 计算 P(T=1 | X)，即给定细胞特征，其位于肿瘤核心的概率
ps_model = LogisticRegression()
ps_model.fit(X, T)  # X是细胞的基因表达特征
propensity_scores = ps_model.predict_proba(X)[:, 1]

# 2. 匹配
# 对于每个在肿瘤核心的细胞，在基质中找一个特征最相似（PS最接近）的细胞
# 使用 1:1 最近邻匹配，卡钳值设为 0.05
matcher = NearestNeighbors(n_neighbors=1)
# 仅在倾向性评分接近的样本中匹配
matched_pairs = matcher.fit(np.column_stack([propensity_scores[T==0], X[T==0]]))

# 3. 平衡性检查
# 比较匹配后，两组协变量 X 的差异是否 < 0.1 (标准化均值差异)
# 确保除了位置不同，其他特征在统计上是一致的

# 4. 估计处理效应 (ATE)
# 在匹配后的样本上，比较 Y (耗竭评分) 的差异
ATE = mean(Y_matched_Treated) - mean(Y_matched_Control)
```

### 统计学新颖性（190字）

本研究引入了**因果推断中的 Rubin因果模型**框架。传统生物学的差异分析是 $E[Y|T=1] - E[Y|T=0]$，这混杂了选择效应。本研究估计的是 $E[Y(1) - Y(0)]$，即同一个细胞在两个不同位置下的潜在结果差异。通过PSM，我们构建了一个**准实验设计**，使得观察性数据接近于随机对照试验（RCT）的严谨性。这是对“位置决定命运”这一生态学观点的统计学矫正。

### 预期突破（240字）

**旧范式的局限**：
- 简单差异分析：无法区分T细胞是因为“累了”才停在基质，还是因为在“基质”才变“累”。
- 伪相关：认为某些基因是“空间特异”的，其实只是“特定亚群”的标志物。

**我的颠覆性**：
1.  **解耦“先天”与“后天”**：区分T细胞的哪些特征是进入肿瘤前就有的（克隆型决定），哪些是进入后被环境诱导的（位置决定）。
2.  **重新定义“耗竭”**：发现所谓的“耗竭T细胞”可能只是一群尚未进入肿瘤的“初发T细胞”，颠覆了现有的免疫治疗靶点选择逻辑。

### 临床/转化价值（210字）

1.  **靶点真实性验证**：只有被证明是由空间位置（肿瘤微环境）诱导的基因，才是可靠的CAR-T或联合治疗靶点。如果差异是先天克隆型导致的，那么靶向该通路可能无效。
2.  **预后模型优化**：基于“纯空间效应”构建的预后指数，将比基于原始转录组的指数更准确。
3.  **联合用药策略**：如果PSM分析显示位置效应主要由缺氧驱动，则联合使用HIF抑制剂是合理的；如果主要由抗原负荷驱动，则需增强抗原释放。

### 数据需求（180字）

- **配对scRNA-seq**：用于训练PSM模型的特征空间。

- **临床队列**：具有详细病理注释和生存随访的样本。
