### 推演草稿 (Internal Reasoning)

**技术降维分析**：
Paper 1 虽然建立了大规模的CSF pQTL图谱，但它存在一个根本性的"维度缺失"问题：**CSF是混合物**。
- **现有局限**：Paper 1将CSF作为一个均质流体进行分析。如果某个pQTL显著，我们不知道这个蛋白是从哪里来的？是神经元释放的？小胶质细胞？还是星形胶质细胞？甚至是血脑屏障（BBB）渗漏导致的血浆污染？
- **认知盲区**：AD是细胞类型非特异性疾病。不同细胞类型（如小胶质细胞 vs 神经元）对AD的贡献截然不同。如果不解析细胞起源，就无法理解机制。

**为什么贝叶斯分层模型是降维打击**：
- **共享编码模型**：我们可以假设CSF中的蛋白丰度是其来源细胞类型中基因表达加权和的函数。
- **数学原理**：
  $Y_{CSF} = \sum_{k} (w_k \cdot X_{k}) + \epsilon$
  其中 $Y_{CSF}$ 是CSF蛋白水平，$X_k$ 是细胞类型 $k$ 的表达，$w_k$ 是权重（分泌率/透过BBB的比率）。
- **推断逻辑**：如果我们有**脑eQTL数据**（如ROSMAP）作为先验，我们可以通过贝叶斯模型推断CSF pQTL最可能起源于哪种细胞类型的eQTL。

当前CSF蛋白质组学研究存在严重的"空间异质性缺失"。Paper 1等顶级工作虽然鉴定了大量pQTLs，但完全忽略了CSF蛋白的**细胞起源**问题。AD涉及神经元、小胶质细胞、星形胶质细胞等多种细胞的相互作用，如果不解析CSF蛋白究竟来自哪种细胞，我们就无法将遗传变异与具体的**细胞病理机制**联系起来。例如，一个位于TREM2基因座的pQTL，究竟是因为小胶质细胞分泌增多，还是因为神经元损伤导致释放？本研究旨在利用**贝叶斯分层模型**，整合脑组织单核eQTL数据，将bulk CSF pQTLs信号解耦为细胞类型特异性的遗传调控信号，从而在无需CSF单细胞数据的情况下，绘制"细胞类型分辨率的CSF遗传调控图谱"。

**贝叶斯分层模型构建**：
构建**细胞类型解耦模型**，其核心思想是认为CSF蛋白水平是各细胞类型基因表达的加权和：
$$ P_{ij} = \sum_{k=1}^{K} \beta_{jk} \cdot G_{ij} + \epsilon_{ij} $$
其中：
- $P_{ij}$ 是个体 $i$ 的蛋白 $j$ 在CSF中的水平。
- $G_{ij}$ 是基因型。
- $\beta_{jk}$ 是蛋白 $j$ 在细胞类型 $k$ 中的效应大小。
- 关键创新：引入**稀疏先验**，如 Spike-and-Slab 先验或 Horseshoe 先验，强制大多数 $\beta_{jk}$ 为0，只保留少数非零值，从而实现"细胞类型特异性"的识别。

import pymc3 as pm

with pm.Model() as cell_deconv_model:
    # 先验：细胞类型活性
    # 利用Horseshoe先验诱导稀疏性，自动选择最相关的细胞类型
    tau = pm.HalfCauchy('tau', beta=2)
    lam = pm.HalfCauchy('lam', beta=2, shape=K_cell_types)
    c = pm.HalfCauchy('c', beta=2)
    
    # 细胞类型特异性效应
    beta_cell = pm.Normal('beta_cell', mu=0, sigma=tau * lam * c, shape=(K_proteins, K_cell_types))
    
    # 似然函数：CSF蛋白水平
    mu = pm.math.dot(Genotypes, beta_cell) # Genotypes: [N_samples, N_SNPs]
    
    # 观测数据（CSF Protein Level）
    sigma = pm.HalfNormal('sigma', sigma=1)
    likelihood = pm.Normal('likelihood', mu=mu, sigma=sigma, observed=CSF_Protein_Level)

## 【假设 3：跨界盲区模式 - 经济学/计量经济学驱动】

### 假设标题
合成控制法构建个体化"反事实"基线：从流体组学动态轨迹中破译阿尔茨海默病的个性化因果干预窗口

### 裂缝模式
跨界盲区

### 学术三要素评估标准 (Academic Three Elements)
- **新颖性**：首次将计量经济学中的**合成控制法**引入生物医学研究，为每个AD患者构建"如果没有发生病理改变"的**反事实基线**，解决临床队列中缺乏对照的难题。
- **严谨性**：通过加权组合多个未患病个体的纵向数据，完美匹配患病者的前期轨迹，确保因果推断的内部有效性。
- **颠覆性**：抛弃传统的"群体平均效应"，转向**个体化因果效应**估计，识别每个人独特的"转折点"。

### 推演草稿 (Internal Reasoning)

**跨界盲区分析**：
Paper 1 使用了横断面的PWAS和静态的遗传分析，这完全忽略了**时间维度**和**个体异质性**。
- **医学现状**：在AD临床试验中，我们很难为每个患者找到一个完美的"对照"。传统RCT是将患者随机分组，但这只能估计"平均处理效应（ATE）"。
- **经济学范式**：在评估某个国家的经济政策效果时，经济学家面临"无法同时观察到同一个国家在实施政策和不实施政策下的经济表现"的难题。他们发明了**合成控制法**。
- **跨界应用**：AD的发展是一个漫长的纵向过程。对于某个特定的患者（如转换为AD的MCI患者），我们可以从"稳定MCI人群"中合成一个**虚拟 twin**。这个虚拟 twin 在转换前的CSF蛋白轨迹与真实患者高度一致，但在转换期保持稳定。两者的差异就是**疾病的因果效应**。

**核心逻辑**：
1. **数据驱动对照**：不需要配对，而是用算法合成。
2. **动态轨迹**：利用Paper 1中提及的"预测模型"，我们不仅关注预测准确率，更关注轨迹的**偏差**。

### 核心科学问题（260字）

当前AD生物标志物研究严重依赖群体水平的横断面分析，忽略了疾病进展的**纵向动态异质性**。Paper 1虽然建立了预测模型，但本质上仍是静态的。核心问题在于：对于某个特定的个体，我们如何区分"正常的衰老相关的蛋白变化"与"AD病理驱动的异常变化"？传统的病例对照研究无法回答这个问题，因为AD患者和对照组在基线就可能存在系统性差异。本研究旨在引入计量经济学的**合成控制法**，为每个AD患者构建一个个性化的"反事实自我"——即如果该个体未患AD，其CSF蛋白组学轨迹本应如何演变。通过比较真实轨迹与合成轨迹的**动态偏差**，精准识别个体化的病理转折点和因果干预窗口。

### 创新假设（200字）

我假设：AD的病理转折并非瞬间发生，而是存在一个**因果累积期**。具体假设为：
1. **轨迹分离假设**：在临床诊断为AD前的3-5年，患者的CSF蛋白轨迹与其"合成对照"将出现显著分离。分离点即为"因果爆发点"。
2. **异质性窗口假设**：不同患者的"因果爆发点"对应的蛋白靶点不同。例如，A患者以神经炎症蛋白（如TREM2）的偏离为起点，而B患者以突触蛋白（如Neurogranin）的下降为起点。
3. **干预效应假设**：在"因果爆发点"之前的干预具有完全的因果效应（可逆转轨迹），而在之后的干预效果呈指数级衰减。

### 技术路线（600字）

**数据准备**：
1. **纵向队列**：ADNI (Alzheimer’s Disease Neuroimaging Initiative) 的CSF蛋白组数据（包含多个时间点：Baseline, M6, M12, M24...）。
2. **分组**：Converter组（MCI转AD），Stable组（MCI长期未转），Control组（NC）。

**合成控制法**：
对于Converter组中的每一个个体 $i$：
1. ** donor pool **：Stable组和Control组中的所有个体。
2. **特征匹配**：在转换前的基线期，匹配一系列CSF蛋白特征（如Aβ42, p-Tau, 以及Paper 1中的38个因果蛋白）。
3. **权重优化**：寻找一组权重 $W = (w_1, w_2, ..., w_N)$，使得Donor的加权组合在转换前的轨迹与个体 $i$ 最大化相似（最小化MSE）。
   $$ \min_{W} || X_{1, i} - X_{0} W || $$
   其中 $X_{1, i}$ 是个体 $i$ 转换前的特征向量，$X_{0}$ 是Donor池的特征矩阵。
4. **反事实推断**：将优化出的权重 $W$ 应用到转换后的时间段，生成"合成个体"的轨迹。
5. **因果效应估计**：
   $$ \tau_{it} = Y_{1, it} - \sum w_j Y_{0, jt} $$
   其中 $\tau_{it}$ 是个体 $i$ 在时间 $t$ 的因果效应（真实值 - 合成值）。

**时间序列因果推断**：
1. **变点检测**：使用**CUSUM (Cumulative Sum Control Chart)** 检测 $\tau_{it}$ 何时显著偏离0。
2. **显著性检验**：通过Permutation test（随机置换Donor池）计算p值，排除偶然性。

```python
# 伪代码：合成控制法实现
from scipy.optimize import minimize

class SyntheticControl:
    def __init__(self, treated_unit, donor_pool, pre_period):
        self.treated = treated_unit # 转换者的数据矩阵 [time, features]
        self.donors = donor_pool    # 对照池的数据矩阵 [n_donors, time, features]
        self.pre_period = pre_period # 切割点索引

    def fit(self):
        # 仅使用转换前的数据进行优化
        X_pre_treated = self.treated[:self.pre_period]
        X_pre_donors = self.donors[:, :self.pre_period, :]

        def loss_function(weights):
            synthetic = np.dot(weights.T, X_pre_donors)
            return np.sum((X_pre_treated - synthetic)**2)

        # 约束：权重和为1，且非负
        constraints = ({'type': 'eq', 'fun': lambda w: np.sum(w) - 1})
        bounds = [(0, 1) for _ in range(self.donors.shape[0])]
        
        result = minimize(loss_function, x0=np.ones(len(self.donors))/len(self.donors),
                          bounds=bounds, constraints=constraints)
        self.weights = result.x

    def predict(self):
        # 预测整个时间段的反事实轨迹
        self.synthetic_trajectory = np.dot(self.weights, self.donors)
        return self.synthetic_trajectory

    def causal_effect(self):
        return self.treated - self.synthetic_trajectory
```

## 【假设 4：降维打击模式 - 网络因果驱动】

### 假设标题
因果网络解耦识别"枢纽因果蛋白"：从全基因组关联到蛋白质互作网络的因果涌现

### 裂缝模式
降维打击

### 学术三要素评估标准 (Academic Three Elements)
- **新颖性**：结合**网络因果推断**与**中介分析**，将孤立的pQTLs连接成动态网络，识别"上游主调节蛋白"。
- **严谨性**：使用**双向孟德尔随机化**排除反向因果，使用**网络传播模型**估计间接效应。
- **颠覆性**：挑战单蛋白药物研发模式，提出"网络靶向"策略，即干预网络中的"枢纽节点"比干预边缘节点更有效。

### 推演草稿 (Internal Reasoning)

**技术降维分析**：
Paper 1 鉴定了38个独立的"因果蛋白"，但它是将它们**孤立看待**的。生物学中，蛋白不是孤立工作的，它们处于复杂的PPI（蛋白互作）网络中。
- **现有局限**：如果蛋白A和蛋白B都因果影响AD，且A调节B，那么：
  - 情况1：A是驱动者，B是下游（干预A即可，干预B无效）。
  - 情况2：A和B是平行通路（需同时干预）。
- Paper 1的方法无法区分这两种情况。

**为什么网络因果推断是降维打击**：
- 我们需要构建一个**有向因果网络**。
- 使用**两步法MR**：
  1. Step 1: 证明 SNP_A -> Protein_A -> AD (A是因果的)。
  2. Step 2: 证明 Protein_A -> Protein_B -> AD (B是A的中介)。
- 如果Step 2显著，那么B就是"虚假因果"（只是A的代理人），不应作为首选药物靶点。

### 核心科学问题（250字）

当前AD药物靶点筛选存在"原子化"倾向，即孤立地评估每个蛋白的因果效应，而忽略了蛋白质互作网络（PPI）中的**级联效应**。Paper 1列出的38个蛋白可能存在大量冗余：某些蛋白之所以显著，仅仅是因为它们受到了真正的"上游驱动蛋白"的调控。本研究旨在利用**网络因果推断**，构建CSF蛋白组的因果调控网络，区分**"上游主调节因子"**和**"下游效应因子"**。核心问题是：如何在网络层面解耦因果流，从而识别出那些处于网络核心位置、干预后能产生最大"级联受益"的枢纽靶点？

### 创新假设（200字）

我假设：AD的遗传风险集中在少数几个**"枢纽因果蛋白"**上。具体假设为：
1. **网络中介假设**：Paper 1鉴定的38个蛋白中，至少20个实际上是这5-10个枢纽蛋白的下游中介者。
2. **收敛性假设**：尽管AD的风险突变分布在全基因组各处，但它们的因果效应最终汇聚到同一个核心蛋白网络模块（如"内吞/溶酶体模块"）。
3. **干预效应倍增假设**：干预枢纽蛋白的因果效应（OR）是干预边缘蛋白的3倍以上，因为枢纽调节了整个下游模块的活性。

### 技术路线（600字）

**网络构建**：
1. **骨架网络**：利用STRING数据库或BioPLEX获取高质量的PPI数据。
2. **因果边定向**：
   - 利用**双向孟德尔随机化**确定每条边的方向。
   - 例如：检测 A->B 和 B->A。如果仅 A->B 显著，则定向为 A -> B。
   - 剔除双向因果（可能是反馈回路）或未确定方向的边。

**中介MR分析**：
对于网络中的每一条路径 $A \to B \to AD$：
1. **总效应**：A对AD的MR效应。
2. **直接效应**：在调整了B的表达水平后（多变量MR），A对AD的效应。
3. **间接效应**：总效应 - 直接效应。
4. **中介比例**：间接效应 / 总效应。
   - 若比例 > 50%，则判定B是A的主要功能执行者，但A是上游控制点。

**枢纽识别算法**：
1. **因果介数中心性**：计算每个节点承载的"因果流"总量。
   $$ CBC(v) = \sum_{s \neq v \neq t} \frac{\sigma_{st}(v)}{\sigma_{st}} $$
   其中 $\sigma_{st}$ 是从s到t的最短因果路径数，$\sigma_{st}(v)$ 是经过v的路径数。
2. **因果流出度**：节点直接影响下游节点的能力。
3. **综合打分**：`Hub_Score = w1 * MR_Effect_Size + w2 * CBC(v) + w3 * Out_Degree`

**模拟验证**：
- 使用网络删除模拟：模拟删除某个枢纽蛋白，观察AD风险评分的下降幅度。预期删除枢纽导致的下降幅度远大于删除边缘节点。

### 统计学新颖性（Statistical Novelty）（200字）

本研究在统计学上实现了从**一维因果**到**网络拓扑因果**的升级。
1. **路径分解**：传统的MR分析是"黑盒"，我们通过中介MR将因果效应分解为"直接效应"和"间接效应"，清晰地展示了信号如何在PPI网络中流动。
2. **网络约束推断**：利用网络结构作为先验约束，减少了多重比较校正的负担。例如，如果蛋白A和蛋白B在网络中紧密相连，我们倾向于认为它们属于同一个生物学通路，从而对它们进行通路级别的FDR校正，而非蛋白级别。
3. **反事实网络干预**：利用**do-calculus**框架，计算"如果我们干预蛋白A，整个网络状态的分布将如何改变"。这是对药物作用机制的数学化描述。
