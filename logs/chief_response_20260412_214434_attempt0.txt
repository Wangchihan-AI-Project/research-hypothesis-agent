## 【假设 2：降维打击模式】

### 假设标题
超越高斯图模型：基于非参数动态贝叶斯网络与物理信息神经网络的空间转录组长程依赖建模

### 裂缝模式
降维打击

### 学术三要素评估标准
- **新颖性**：首次引入流体力学中的物理约束来建模细胞间的信号传播。
- **严谨性**：物理信息神经网络（PINN）确保模型符合生物学扩散定律，拒绝黑盒拟合。
- **颠覆性**：将离散的Spot分析转化为连续场的物理问题，解决稀疏性问题。

### 推演草稿

**技术降维分析**：
空间转录组分析的现有方法存在严重的“降维”缺陷：
1. **Seurat/Scanpy**：将Spot视为高维向量，用PCA或UMAP降维。这**完全丢失了物理空间的连续性**，且假设数据符合高斯分布，这在生物学中是错误的（基因表达通常是零膨胀的）。
2. **Squidpy/Giotto**：基于邻域的统计量（如Moran's I），仅能捕获**局部（一阶）相关性**。这忽略了细胞因子通过扩散或外泌体运输形成的**长程依赖**（例如，缺氧核心释放的VEGF可以影响数百微米外的血管生成）。
3. **SpaGCN**：利用图卷积网络（GCN），但GCN本质上是低通滤波器，**过度平滑**了局部特征，导致无法区分精细的亚结构。

**核心问题**：肿瘤微环境是一个**非平稳随机场**。目前的AI模型只是在做“模式识别”，而没有建模“物理机制”。

**降维打击 - 物理信息神经网络 (PINN)**：
我假设将空间转录组看作是一个**物理扩散过程**的快照：
- 配体（如TGF-β）从源细胞产生，遵循**反应-扩散方程**在组织间传播。
- 我们在空间上观测到的基因表达，是扩散源、扩散系数（组织屏障）和降解率的函数。

使用PINN，我们可以：
1. 在损失函数中加入**物理定律**（如 $\frac{\partial u}{\partial t} = D \nabla^2 u + f(u)$）。
2. 即使在Spot缺失的区域，物理定律也能约束预测，实现**零样本学习**。

### 核心科学问题

当前空间转录组分析受限于“数据驱动”的黑盒模型，完全忽略了组织切片中的物理化学约束。主流方法（如GCN、SpatialDM）仅能基于共表达推断局部细胞通讯，无法建模跨越数百微米的梯度驱动的长程依赖，且在Spot分辨率不足（如55μm直径）时，因严重的稀疏性导致推断失效。本研究假设肿瘤微环境的分子分布受**反应-扩散方程**控制，核心问题是：如何利用物理信息神经网络（PINN）将离散的基因表达数据嵌入到连续的物理场中，从而反推出不可见的信号源（如配体分泌细胞）及组织屏障（如基质密度），实现对长程细胞通讯的因果建模？

### 创新假设

我假设：空间转录组数据并非独立的样本点，而是**连续生物物理场**的离散采样。引入物理信息神经网络（PINN），通过**反应-扩散偏微分方程**约束神经网络的训练，使其在拟合基因表达的同时，学习组织内的“扩散系数图”和“源项图”。具体假设为：（1）通过PINN反演得到的“扩散屏障图”与H&E染色中的胶原纤维密度高度相关（Pearson r > 0.7）；（2）基于物理约束的模型在缺失Spot处的插值精度比Gaussian Process高40%以上；（3）识别出的“长程通讯枢纽”不仅是高表达细胞，更是处于特定物理位置（如肿瘤-基质界面）的“拓扑关键点”。

### 抛弃的旧方法（预期突破）

- **旧方法**：基于GCN的图神经网络，或基于距离的邻近分析方法。
- **局限性**：只能看邻居，无法看全局；缺乏可解释性，无法解释为什么这个基因在这里表达。
- **我的颠覆性**：
    1. **物理驱动**：不是看谁和谁像，而是看“谁产生的信号能扩散到这里”。
    2. **连续场建模**：填补了Spot之间的空白，实现了亚细胞级别的分辨率虚拟重构。
    3. **机制解耦**：将表达值解耦为“源强度”（生物学因素）与“扩散率”（物理学因素）。

### 技术路线

**物理约束构建**：
1. 定义反应-扩散方程：$\frac{\partial \rho}{\partial t} = \nabla \cdot (D(x) \nabla \rho) + R(x) \rho - \lambda \rho$，其中 $\rho$ 是配体浓度，$D(x)$ 是空间依赖的扩散系数（代表基质密度），$R(x)$ 是分泌率。
2. 假设稳态（$\frac{\partial \rho}{\partial t} \approx 0$），方程简化为 Poisson 方程类型。

**PINN架构**：
```python
# 伪代码：物理信息神经网络
class SpatialPINN(nn.Module):
    def forward(self, x, y):
        # 神经网络拟合配体浓度场
        rho = self.net(torch.cat([x, y])) 
        
        # 自动微分计算物理梯度
        rho_x = torch.autograd.grad(rho, x, grad_outputs=torch.ones_like(rho), create_graph=True)[0]
        rho_y = torch.autograd.grad(rho, y, grad_outputs=torch.ones_like(rho), create_graph=True)[0]
        rho_xx = torch.autograd.grad(rho_x, x, ...)
        rho_yy = torch.autograd.grad(rho_y, y, ...)
        
        # 物理残差 (Loss Physics)
        # D * Laplacian(rho) + Source - Decay = 0
        loss_physics = torch.mean((self.D * (rho_xx + rho_yy) + self.source - self.decay * rho)**2)
        
        # 数据拟合残差 (Loss Data)
        loss_data = torch.mean((rho - self.observed_expression)**2)
        
        return loss_data + lambda_physics * loss_physics
```

**解耦分析**：
1. 训练后，$D(x)$ 代表组织通透性（基质屏障），$Source(x)$ 代表细胞分泌活性。
2. 对比 $D(x)$ 与病理图像分割结果，验证物理意义。

**因果推断**：
- 使用**do-calculus**干预：在模拟中固定 $D(x)$，改变 $Source(x)$，观察远处浓度的变化，从而量化长程因果效应。

### 统计学新颖性

本研究的统计学核心在于将**偏微分方程（PDE）的正反演问题**引入组学数据分析。
1. **克服维数灾难**：传统的空间统计量（如Kriging）在计算协方差矩阵时面临 $O(N^3)$ 的复杂度。PINN利用神经网络的逼近能力，将复杂度降至与样本量线性相关，且能处理非结构化网格。
2. **物理约束的正则化作用**：PDE Loss充当了极强的先验知识。在基因表达数据稀疏（Dropout）的情况下，物理定律填补了信息缺失，这比任何数学上的正则化（L1/L2）都更具生物学合理性。
3. **因果层面的可解释性**：通过识别PDE参数（如扩散系数 $D$），我们将统计相关性转化为物理机制。例如，若某区域 $D$ 极低，意味着存在物理屏障，即使信号源很强，远处也无法接收。这种解释无法通过相关性分析得出。

### 临床/转化价值

1. **物理屏障评估**：量化肿瘤基质的致密程度（“纤维化评分”），预测药物递送的难易程度。
2. **免疫排斥机制解释**：直观展示“物理屏障”还是“缺乏趋化因子”导致了免疫细胞无法浸润。
3. **辅助病理诊断**：在传统的H&E染色之外，提供一张“功能物理图”，显示组织内的压力和信号流动。

### 前沿框架

物理信息神经网络 + 反应-扩散方程 + 因果反演

## 【假设 3：跨界盲区模式】

### 假设标题
双重机器学习驱动的异质性工具变量：破解空间转录组中的未测量混杂与空间自相关陷阱

### 裂缝模式
跨界盲区

### 学术三要素评估标准
- **新颖性**：引入计量经济学中的**双重机器学习**框架，解决高维组学数据中的内生性问题。
- **严谨性**：对空间自相关进行严格的统计控制，拒绝无效的独立性假设。
- **颠覆性**：证明许多已发表的空间基因关联是空间自相关导致的虚假相关。

### 推演草稿

**跨界盲区分析**：
**计量经济学**与**生物信息学**在面对数据时有一个根本性的认知差异：
- **生物信息学**：默认“近邻”即“因果”。如果配体L和受体R在空间上共定位，就认为有互作。
- **计量经济学**：极度恐惧**遗漏变量偏倚（OVB）**和** simultaneity（联立性）**。在空间数据中，这表现为**空间自相关**。

**盲区所在**：
传统的空间统计（如SpatialDE、SPARK）使用高斯过程或广义线性模型（GLM），但这些方法假设误差项是独立的。
然而，空间转录组数据具有极强的**空间自相关性**。
- 问题1：如果一个基因在肿瘤核心高表达，另一个基因也在肿瘤核心高表达，仅仅是因为它们都在“肿瘤核心”这个区域，而不是因为它们有功能互作。这叫**虚假空间显著性**。
- 问题2：我们无法测量所有的混杂因素（如缺氧梯度、pH值、基质压力）。

**跨界引入 - 双重机器学习**：
DML (Chernozhukov et al., 2018) 是解决高维控制变量和内生性问题的终极武器。
我假设使用**空间滞后项**或**基因的多态性（eQTL）**作为工具变量，结合DML框架，在控制了成千上万个其他基因表达（作为混淆因素）后，识别出**稳健的、空间特异性的因果基因**。

### 核心科学问题

空间转录组学研究中存在一个巨大的“跨界盲区”：现有方法将空间共定位直接等同于因果互作，却忽略了**空间自相关**引起的结构式虚假相关。正如经济学中的“价格与销量”受宏观经济因素影响一样，空间中的基因表达受区域效应（如肿瘤核心vs边缘）的深度混杂。目前的GLM或GP模型无法处理数万个基因作为高维混淆变量的情况。本研究的核心问题是：如何借用计量经济学中的**双重机器学习**和**异质性工具变量**，在控制空间自相关和高维混淆的前提下，准确推断局部基因表达对周围微环境的因果效应（LATE）？

### 创新假设

我假设：空间转录组中的许多显著相关性实际上是**空间遗漏变量偏倚**的产物。引入双重机器学习框架：
1. **第一阶段**：使用深度神经网络（ResNet）在高维基因空间中预测“处理变量”（如配体L表达），并预测“结局变量”（如受体R表达），剔除混淆因素。
2. **残差回归**：仅在残差上进行因果效应估计。
3. **工具变量**：利用该Spot在组织切片中的**几何坐标特征**（如距离血管的距离、曲率）作为空间工具变量，因为这些几何特征由发育过程决定，独立于当下的基因表达噪声。
具体假设为：（1）使用DML校正后，现有文献中报道的40%的配体-受体对将不再显著；（2）识别出的因果基因具有更强的进化保守性；（3）LATE（局部平均处理效应）显示出明显的空间异质性，即同一配体在不同区域（核心vs边缘）具有截然相反的因果效应。

### 抛弃的旧方法（预期突破）

- **旧方法**：SpatialDE、SPARK、CellPhoneDB。
- **局限性**：假设残差独立同分布，忽略了空间滞后；无法处理高维混淆变量（容易过拟合）。
- **我的颠覆性**：
    1. **正交化**：将预测问题与因果推断解耦，防止正则化偏差。
    2. **混淆控制**：可以同时控制10,000个基因的表达作为协变量，这是传统回归做不到的。
    3. **因果识别**：不是预测谁在一起，而是预测“如果改变X，Y会如何变化”。

### 技术路线

**DML框架构建**：
1. **数据**：10x Visium数据（n=100,000 spots）。
2. **模型**：
   - $g_L(Z)$：使用神经网络预测配体 $L$ 的表达（基于所有其他基因 $Z$）。
   - $g_R(Z)$：使用神经网络预测受体 $R$ 的表达（基于所有其他基因 $Z$）。
   - 获取残差：$V = L - g_L(Z)$, $W = R - g_R(Z)$。
   - **因果效应**：$\theta = \frac{E[V \cdot W]}{E[V^2]}$。

**空间工具变量**：
- 构建外生变量：Spot的拓扑特征（Degree centrality, Betweenness in the spatial graph）。
- 假设：拓扑结构影响配体分布（相关性），但不直接影响受体的转录反应（外生性），除非通过配体。
- **2SLS**：第一阶段用拓扑特征预测配体，第二阶段用预测值解释受体。

**统计推断**：
- 使用**Neyman正交化**确保因果效应估计的渐近正态性。
- 计算**Confidence Interval**，进行严格的统计显著性检验。

### 统计学新颖性

本研究将经济学诺贝尔奖级别的工具（DML）引入生物医学，解决了组学数据的“维数诅咒”与“内生性”双重难题。
1. **Neyman正交化**：传统的机器学习正则化（如Lasso/Ridge）会引入偏差，导致因果效应估计有偏。DML通过“正交得分”保证了即使第一步的机器学习模型有误差，最终的因果估计仍然是无偏的。这是统计学上的巨大飞跃。
2. **异质性处理效应**：不同于传统线性回归估计的平均效应（ATE），DML可以估计每个Spot或每个亚群的个体处理效应（ITE），揭示“在特定微环境下才有效”的因果机制。
3. **无效性检验**：利用F-statistic严格检验工具变量的强度，拒绝Weak Instrument导致的不可靠推断，这在目前的生物信息学分析中常被忽视。

### 临床/转化价值

1. **可靠靶点筛选**：剔除因环境共表达导致的假阳性靶点，降低药物研发失败率。
2. **精准给药策略**：基于LATE的空间异质性，指导“局部给药”或“分区治疗”，例如仅在核心区域给予特定抑制剂。
3. **生物标志物重估**：重新评估现有的预后标志物，去除那些仅仅是伴随空间位置变化而变化的“乘客基因”。

### 前沿框架

双重机器学习 (DML) + 空间计量经济学 + Neyman正交化
