# 伪代码：门控因果图网络
class CausalGNN(nn.Module):
    def __init__(self):
        self.feature_encoder = GraphConv(in_dim=200, out_dim=64) # 影像特征
        self.confounder_encoder = GraphConv(in_dim=400, out_dim=64) # 邻居特征
        self.gate_mechanism = nn.Sequential(nn.Linear(128, 1), nn.Sigmoid())

    def forward(self, node_features, adj_matrix):
        # 1. 提取节点自身特征
        h_self = self.feature_encoder(node_features)
        # 2. 聚合邻居特征（作为混淆项）
        h_neighbor = self.confounder_encoder(node_features, adj_matrix)
        # 3. 门控机制：学习邻居对当前节点的"影响权重"
        gate = self.gate_mechanism(torch.cat([h_self, h_neighbor], dim=-1))
        # 4. 残差连接：解耦后的纯净表征
        h_pure = h_self + (1 - gate) * h_neighbor
        return h_pure, gate
```

        # 2. 强制解耦为三个独立分量

        self.z_state = z[:, 10:60]   # 假设状态维度
        self.z_conf = z[:, 60:]      # 残差/混淆

    def compute_loss(self, x, x_recon):

        loss_recon = MSE(x, x_recon)

## 【假设 3：跨界盲区模式 - 计量经济学驱动】

### 假设标题
**基于工具变量与断点回归的空间转录组因果解锁：解开微环境免疫细胞的内生性死结**

### 裂缝模式
跨界盲区

### 学术三要素评估标准
- **新颖性**：将计量经济学中的**工具变量**和**断点回归**（RD）首次引入空间转录组学，解决肿瘤微环境（TME）研究的内生性难题。
- **严谨性**：使用**外生冲击**（如血管物理距离）作为工具变量，满足排他性约束，严格剔除混杂因素。
- **颠覆性**：建立"空间因果推断"新范式，证明空间邻近性不等于因果性，推翻传统基于距离的细胞互作结论。

### 推演草稿

**跨界盲区分析**：
在生物信息学领域，空间转录组（ST）分析几乎清一色使用**共表达**或**邻近性**来定义细胞互作：
-   **方法**：计算Spot A（肿瘤）和Spot B（免疫）的距离，或者计算基因表达相关性。
-   **假设**：距离近 = 有互作；相关系数高 = 有调控。

这在计量经济学看来是极其 naive 的，犯了**遗漏变量偏误（OVB）**：
-   **内生性**：肿瘤细胞和免疫细胞都受到微环境中第三因素的驱动（如缺氧、pH值）。我们观察到的"肿瘤-免疫互作"，可能只是它们同时对缺氧做出的反应（虚假相关）。
-   **双向因果**：是肿瘤招募了巨噬细胞？还是巨噬细胞促进了肿瘤生长？传统相关分析完全无法判断方向。

**我的切入点 - IV与RD跨界**：
-   **工具变量（IV）**：寻找一个**只影响免疫细胞聚集，但不直接影响肿瘤基因表达**的变量。
    -   候选IV：**三级淋巴结构（TLS）的物理边界**。TLS是免疫细胞的"大本营"，免疫细胞的分布受TLS位置（外生变量）的强烈约束。
    -   逻辑：TLS在哪里，免疫细胞就被迫在哪里。这切断了"免疫细胞 -> 肿瘤"的反向因果路径。
-   **断点回归（RD）**：利用组织病理学特征（如血管壁）作为"断点"。比较血管内外的基因表达突变。
    -   假设：在血管壁两侧50μm内，细胞类型应当是连续的，但基因表达受氧浓度影响发生跳跃。这个"跳跃"就是因果效应。

### 核心科学问题（280字）

肿瘤微环境（TME）的细胞互作研究长期陷入"内生性"泥潭。目前的分析方法（如CellChat, Giotto）默认空间邻近性等同于功能性互作，但这无法排除混杂因素（如缺氧梯度、组织结构硬边界）的干扰。例如，免疫细胞浸润丰度与肿瘤恶性程度高度相关，但这可能是因为缺氧环境同时促进了肿瘤侵袭和免疫细胞趋化，而非二者存在直接互作。这种**遗漏变量偏倚（OVB）**导致大量无法重复的假阳性发现。本研究的核心问题是：如何借鉴计量经济学的**工具变量（IV）**和**断点回归（RD）**设计，利用空间转录组的空间结构特征，构建外生冲击变量，从而在观察性数据中识别肿瘤与免疫细胞间的**纯因果效应**？

### 创新假设（220字）

我假设：肿瘤微环境的空间结构特征（如三级淋巴结构TLS的中心、血管壁、坏死区边缘）构成了天然的**自然实验**。具体假设为：（1）**TLS中心距**可以作为免疫细胞浸润的工具变量（IV），满足相关性和排他性约束（F-statistic > 10）；（2）基于IV的两阶段最小二乘法（2SLS）分析显示，传统方法高估了肿瘤-免疫互作效应达50%以上；（3）利用**模糊断点回归（Fuzzy RD）**分析血管边界，发现缺氧诱导因子（HIF1a）在跨越血管临界点（<100μm）时存在显著的因果跳跃（Jump > 2 logFC），揭示了氧气梯度的因果调控。

### 技术路线（650字）

**数据与预处理**：
1.  **数据源**：10x Visium FFPE数据（乳腺癌、肺癌），Stereo-seq数据（高分辨率）。
2.  **图像配准**：将H&E图像与转录组Spot严格对齐。
3.  **结构提取**：使用ResNet-50从H&E图像中分割"血管"、"TLS"、"坏死区"作为结构特征。

**工具变量策略（2SLS）**：
1.  **第一阶段（回归内生变量）**：
    -   内生变量：免疫细胞丰度。
    -   工具变量：距离最近TLS中心的距离 ($D_{TLS}$)。
    -   方程：$Immune_i = \pi_0 + \pi_1 D_{TLS} + \text{Controls} + u_i$
    -   验证：检查F-statistic，确保工具变量强度。
2.  **第二阶段（回归结局）**：
    -   结局变量：肿瘤细胞的恶性基因表达（如MKI67, EMT signature）。
    -   方程：$Malignancy_i = \beta_0 + \beta_1 \widehat{Immune}_i + \text{Controls} + e_i$
    -   解读：$\beta_1$ 即为剔除了混杂因素后的因果效应。

**断点回归策略（RD）**：
1.  **断点定义**：以血管壁为0点，定义驱动变量（Running Variable）为距离血管壁的距离。
2.  **模糊断点设计**：
    -   由于细胞分布不完全精确在边界上，使用Fuzzy RD。
    -   局部多项式回归：拟合距离与基因表达的关系。
    -   检验：在边界处（$\lim_{d \to 0^+}$ vs $\lim_{d \to 0^-}$）是否存在跳变。
3.  **RD有效性检验**：
    -   **McCrary密度检验**：验证个体（细胞）无法精确操纵其位置（即细胞不能自主决定长在血管左边还是右边，满足随机化假设）。
    -   **安慰剂检验**：在血管内500μm处人为制造假断点，检验结果是否显著（应不显著）。

**空间异质性控制**：
-   使用**地理加权回归（GWR）**捕捉空间非平稳性。
-   纳入经纬度坐标作为固定效应，控制不可观测的空间混杂。

### 统计学新颖性（200字）

本研究将计量经济学的**因果推断金标准**移植到了生物组学领域。（1）**工具变量（IV）**：这是解决内生性问题的终极武器。我们巧妙地利用解剖学结构（TLS）作为"自然随机化装置"，因为免疫细胞受解剖结构限制而被"分配"到特定位置，这模仿了随机对照试验（RCT）的逻辑。（2）**断点回归（RD）**：利用组织物理边界（如血管壁）产生的剧烈环境跳变，识别局部平均处理效应（LATE）。这种设计在内部效度上仅次于RCT。（3）**排他性约束检验**：严格验证工具变量仅通过"免疫浸润"影响"肿瘤状态"，而非通过其他路径（如血管分泌因子）。这种严密的逻辑链条在目前的Nature/Cell级别的生物论文中也是极其罕见的。

### 预期突破（280字）

**旧方法局限性**：
- 空间统计：仅计算Moran's I或Geary's C，描述相关性，无法推断因果。
- NicheNet：基于先验知识库，无法发现新机制，且受限于配体-受体数据库的完整性。
- 伪空间分析：简单的差异分析，忽略空间位置信息。

**我的颠覆性**：
1.  **剔除虚假相关**：证明大量被传统方法认为是"互作"的信号，实际上只是空间共定位（Colocalization）。
2.  **量化因果剂量**：首次在组织切片尺度上给出"每增加1个免疫细胞，肿瘤恶性风险下降多少"的因果估计值。
3.  **定位因果边界**：利用RD设计，精确定位微环境中物理或化学屏障（如基质边界）的"因果作用范围"（例如，距离血管<100μm是有效杀伤区，>200μm是免疫沙漠）。
4.  **范式转移**：从"描述性空间生物学"转向"因果性空间生物学"。

### 临床/转化价值（230字）

本研究将为免疫治疗提供全新的空间生物标志物：
1.  **疗效预测**：基于因果效应量（而非单纯的细胞丰度）预测PD-1响应率。只有那些具有"高因果杀伤效应"的TLS才具有预测价值。
2.  **给药策略**：根据"因果作用范围"（如RD发现的100μm有效半径），设计药物递送系统（如纳米颗粒），确保药物能渗透到因果边界之外。
3.  **联合治疗**：如果IV分析发现缺氧是主要的混杂因素，则联合抗血管生成药物（正常化血管）可能比单纯免疫检查点阻断更有效。

### 数据需求（180字）

-   **数据集**：10x Visium高分辨率数据（包含H&E图像）。
-   **临床数据**：患者的免疫治疗响应队列（ responders vs non-responders）。

-   **先验知识**：已知的空间解剖结构坐标（TLS、血管、坏死区）。
