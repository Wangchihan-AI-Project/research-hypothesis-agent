#### 研究背景
我们提出一个面向新辅助免疫治疗早期疗效判别的研究假设：在治疗窗口期内，肿瘤组织液中的局部免疫因子动态变化模式，可能比单次组织活检或外周血静态指标更早反映后续病理缓解趋势。该假设希望结合新辅助治疗前后多时间点组织液样本、配对病理评估结果及基础临床资料，识别能够表征局部免疫激活与免疫抑制再平衡过程的动态特征组合。核心目标是验证“组织液局部动态信号”是否值得作为后续深入开发的早期反应表征，而不是立即给出成熟的临床实施方案。

### 【2. Methods - 方法论】

#### 验证策略

采用探索性发现队列与后续小规模前瞻验证队列的两阶段设计。第一阶段用于识别与病理缓解相关的组织液动态特征组合，第二阶段用于检验这些特征在真实窗口期场景中的方向一致性和分层能力。主要终点为病理缓解相关分层表现，次要终点为与影像学趋势和术后复发风险指标的一致性。

#### 技术保障

| 内容 |
|------|
| 按患者维度组织多时间点样本，避免将同一患者的窗口期重复观测误当作独立样本。 |
| 候选免疫因子组合、时间窗定义和动态指标构建仅在发现集完成，再固定至验证集。 |
| 病理缓解标签与组织液指标分析过程分离，减少结果导向的特征筛选偏差。 |

#### 偏倚控制

分析中控制肿瘤类型、给药方案、取样时间偏差、组织液采集方式和样本处理时间差异；并对低体积样本、炎症干扰、局部坏死区域影响和缺失时间点样本进行敏感性分析，避免将采样过程噪声误判为真实动态信号。
