### 【1. Hypothesis - 假设】

#### 标题
**基于肿瘤引流淋巴结单细胞图谱与ctDNA动态联合建模的免疫治疗超早期获益判别假设** (_v1_)

#### 研究背景
我们提出一个面向实体瘤免疫治疗超早期获益识别的研究假设：在治疗启动后数周内，肿瘤引流淋巴结中的树突细胞成熟状态、T 细胞初始激活轨迹与B 细胞辅助反应重构，会与外周 ctDNA 清除速度形成可耦合的早期信号组合，并先于常规影像学评估反映患者是否可能获得持续获益。该假设希望结合肿瘤引流淋巴结单细胞转录组/免疫受体测序、同期外周血 ctDNA 动态监测以及基础临床结局，建立一个用于超早期获益判别的生物学分层框架。核心目标不是立即形成临床检测产品，而是验证“淋巴结免疫启动质量 + ctDNA 早期下降模式”是否构成稳定、可重复且具有跨队列方向一致性的前沿表征。

### 【2. Methods - 方法论】

#### 验证策略

采用发现队列 + 外部验证队列的两阶段设计。发现队列用于识别与持续获益相关的淋巴结免疫激活模式及其与 ctDNA 动态的联合指标；外部验证队列仅用于检验效应方向、风险分层能力和模型稳定性。主要终点为治疗早期持续获益分层表现，次要终点为与持久缓解率和 PFS 趋势的一致性。

#### 技术保障

| 内容 |
|------|
| 以患者级划分发现集与验证集，避免同一患者多时间点血样或淋巴结样本跨集合泄露。 |
| 单细胞聚类、批次校正、状态注释和候选 signature 提取仅在发现集内完成，再冻结后应用到验证集。 |
| ctDNA 清除相关指标采用预先定义的时间窗和变化率公式，禁止根据结局事后挑选最佳阈值。 |

#### 偏倚控制

分析中显式控制肿瘤类型、治疗方案、基线肿瘤负荷、淋巴结取样方式和采样时间偏差；同时对低质量细胞、局部炎症干扰、ctDNA 低丰度样本和样本缺失模式进行敏感性分析，避免将技术噪声或取样偏倚误判为真实的免疫启动信号。
