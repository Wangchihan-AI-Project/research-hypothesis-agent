#### 研究背景
我们提出一个面向实体瘤免疫治疗早期反应识别的研究假设：治疗启动后短期内，肠道菌群代谢物谱的变化与外周 T 细胞疲劳相关标志的动态联动，可能比单独观测菌群组成或外周免疫表型更早提示患者后续反应趋势。该假设希望结合治疗前后粪便代谢组检测、外周免疫表型测量和基础临床结局，构建一个用于早期反应预测的联合生物学框架。核心目标是验证这类跨系统动态信号是否具备稳定的预测潜力，而不是立即提出成熟的机制闭环或临床转化路径。
